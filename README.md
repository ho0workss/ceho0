# ceho0 — 종목 추천 대시보드

뉴스 · 차트 · 펀더멘털 지표와 몬테카를로 시뮬레이션을 결합한 기간별(당일/1주/1개월/장기) 종목 추천 웹앱.
요청 → GitHub 커밋 → Vercel 자동 배포 파이프라인으로 운영됩니다.

## 기능

- **기간별 추천**: 당일 / 1주일 / 1개월 / 장기 전략별 종목 (티커 · 매수/매도 시기와 가격 · 손절가)
- **판단 근거**: 뉴스 · 기술적(차트) · 펀더멘털 3개 축으로 명시, 예측 변수(리스크 요인) 별도 제공
- **시뮬레이션**: GBM 몬테카를로 20,000 경로 — 이익확률, 목표/손절 도달확률, 백분위 밴드(팬차트), 수익률 분포(히스토그램)
- **배당 정보**: 1주당 배당금, 배당 일정, 배당수익률, 다음 배당 시기
- **세후 계산기**: 한국 거주자 기준 양도세(해외 22%, 250만원 공제) · 증권거래세(0.15%) · 배당원천징수(15%/15.4%) 반영 실수령액
- **히스토리 자동 기록**: 추천이 생성될 때마다 `data.js`에 배치 단위로 누적 — 과거 배치는 수정하지 않고 보존
- **필터**: 기간 · 시장(미국/한국) · 리스크 · 배당주 여부

## 구조

- `index.html` / `styles.css` / `app.js` — 정적 SPA (빌드 단계 없음, 외부 의존성 없음)
- `data.js` — 추천 배치 데이터 (히스토리 누적)
- `data/sim.js`, `data/sim.json` — 시뮬레이션 결과 (커밋된 산출물)
- `scripts/simulate.mjs` — 몬테카를로 시뮬레이션 생성기: `node scripts/simulate.mjs > data/sim.json`

## 새 추천 배치 추가 절차

1. 최신 시장 데이터 조사 후 `scripts/simulate.mjs`의 `PICKS` 갱신 → 실행해 `data/sim.json`·`data/sim.js` 재생성
2. `data.js`의 `batches` 배열 **맨 앞에** 새 배치 추가 (기존 배치는 수정 금지)
3. 커밋 & 푸시 → Vercel 자동 배포

## 고지

본 프로젝트는 정보 제공 목적의 시뮬레이션 도구이며 투자 자문이 아닙니다. 모든 수치는 확률 추정치로 실제 수익을 보장하지 않습니다.
